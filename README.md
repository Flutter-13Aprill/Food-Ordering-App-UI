## Installation

1.  Clone the repository:

    `git clone https://github.com/your-username/your-repo-name.git` 

2.  Navigate to the project folder:

    `cd your-repo-name` 

3.  Install dependencies:

    `flutter pub get` 

4.  Run the app:

    `flutter run`
